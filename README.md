# 株式会社 竹輝工業 - 企業紹介Webサイト

![CSS3](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white)

創業60年の金属加工会社「株式会社 竹輝工業」の企業紹介サイトです。実在する家族経営の会社を題材に、ヒアリングから実装まで一人で担当しました。

## 概要

事業内容・会社概要・お問い合わせ先をタブ切り替え形式でまとめた、1ページ完結型のサイトです。

## 制作の背景

家族が経営する会社には紹介用のWebサイトがなかったため、実際に使える形での制作を通じて、要件整理からデザイン・実装までの一連の流れを経験したいと考えました。

## 制作プロセス

* 事業内容・掲載したい情報についてヒアリング
* 掲載情報の整理、ページ構成の検討
* 文章作成・デザイン調整
* HTML/CSSでの実装

制作過程ではAIも活用しましたが、掲載内容の事実確認・構成の判断・最終的な修正は自身で行いました。

## 使用技術

| 分類 | 内容 |
|---|---|
| マークアップ | HTML5 |
| スタイル | CSS3(タブ切り替え・レスポンシブ対応・アニメーション背景スライド) |
| スクリプト | JavaScript(タブコンテンツの出し分け) |
| 依存ライブラリ | なし(素のHTML/CSS/JSのみ) |

## 主な機能

* タブ切り替えによる「トップ / 事業案内 / 会社情報 / お問い合わせ」の1ページ構成
* 会社概要・社歴・取引先・設備情報のテーブル表示
* Googleマップの埋め込み
* モバイル対応のレスポンシブレイアウト
